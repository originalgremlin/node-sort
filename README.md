node-sort
=========

An implementation of various sorting algorithms in NodeJS.

http://en.wikipedia.org/wiki/Sorting_algorithm

Implemented:
    Bingo sort
    Bogosort
    Bozosort
    Bubble sort
    Comb sort
    Insertion sort
    Selection sort
    Shell sort
    Stooge sort

In progress:
    Quicksort
    Merge sort

Planned:
    Binary tree sort
    Bucket sort
    Cocktail sort
    Counting sort
    Cycle sort
    Distribution sort
    Heapsort
    Intro sort
    Library Sort
    List insertion sort (skip list?)
    Merge insertion (Ford-Johnson) sort
    Odd-even sort
    Radix sort
    Smoothsort
    Timsort
